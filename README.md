# API Test Project 

### 1. Clone the Repo
```
git clone https://github.com/AyyJimmehh/api-test.git
cd api-test
```

### 2. Install Requirements
```
pip install -r requirements.txt
```

### 4. Train model
```
python train_model.py
```

### 5. Creat .env file
```
API_KEY=your_secret_key
```

### 6. Start the API
````
uvicorn run:app --reload
````
